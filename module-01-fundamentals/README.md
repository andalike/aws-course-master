# Module 01: AWS Fundamentals

## Welcome to Your Cloud Journey!

This module is designed for **absolute beginners** with zero prior cloud experience. By the end of this module, you'll have a solid foundation in cloud computing concepts and hands-on experience with AWS.

| Attribute | Details |
|-----------|---------|
| **Difficulty Level** | Beginner (No prerequisites) |
| **Estimated Time** | 4-6 hours |
| **Hands-on Labs** | 1 |
| **Quiz Questions** | 15 |

---

## Learning Objectives

By completing this module, you will be able to:

1. **Explain** what cloud computing is and why organizations are migrating to the cloud
2. **Differentiate** between IaaS, PaaS, and SaaS service models
3. **Describe** AWS global infrastructure including Regions, Availability Zones, and Edge Locations
4. **Create** and secure an AWS account with MFA and billing alerts
5. **Navigate** the AWS Management Console efficiently
6. **Install** and configure the AWS CLI on your local machine
7. **Understand** AWS Free Tier limits to avoid unexpected charges
8. **Execute** basic AWS CLI commands

## Key Terminology

Before you begin, familiarize yourself with these essential terms:

| Term | Definition |
|------|------------|
| **Cloud Computing** | Delivery of computing services over the internet |
| **AWS** | Amazon Web Services - Leading cloud provider |
| **Region** | Geographic area containing AWS data centers |
| **Availability Zone** | Isolated data center within a Region |
| **Console** | Web-based interface to manage AWS services |
| **CLI** | Command Line Interface for AWS |
| **Free Tier** | AWS services available at no cost for limited usage |
| **IAM** | Identity and Access Management - security service |
